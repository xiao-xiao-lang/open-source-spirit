\# 开源精神分号器



一个完整的开源项目标识符管理系统，基于 Cloudflare Pages + D1 数据库构建。



\## 🚀 功能特性



\- 🔍 \*\*项目查询\*\*：通过开源号或项目名称快速查找开源项目

\- 📝 \*\*申请开源号\*\*：为开源项目申请唯一标识符（KY+年月+随机字符串）

\- 👤 \*\*用户管理\*\*：支持QQ、邮箱、电话、B站链接等多种联系方式

\- 🛡️ \*\*管理员后台\*\*：统一管理员账户，可审核、管理所有项目

\- 📊 \*\*数据统计\*\*：可视化统计图表，实时查看系统状态

\- 📱 \*\*响应式设计\*\*：适配桌面端和移动端



\## 🏗️ 技术架构



\- \*\*前端\*\*：HTML5 + CSS3 + Vanilla JavaScript

\- \*\*后端\*\*：Cloudflare Functions（Serverless API）

\- \*\*数据库\*\*：Cloudflare D1（SQLite）

\- \*\*部署\*\*：Cloudflare Pages（全球CDN）

\- \*\*样式\*\*：自定义CSS + Font Awesome图标
